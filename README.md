# Ares 自定义地基面积
来自 https://github.com/Ares-Developers/Ares-docs 上的原有的工具
位于custom_foundation.html以及jquery-1.4.2.min.js这两个文件
同时感谢Ares人员的源码文档，带来的方便与帮助。

在这里添加了一些内容，在这里现在支持如下：
1.语言切换（英文/中文），现在支持了中文语言。
2.增加了视图按钮，作为（斜面/正面）的切换功能。
3.代码输入框上方增加了复制代码按钮，用于方便操作。
4.右上角使用描述，改写了理解方向，为了更好操作。

## 仓库用途
这个仓库用来储存相应文件以及部署网站，可方便使用。
